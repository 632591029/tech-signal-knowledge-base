### 1. OpenAI 宣布 ChatGPT 将引入广告

**来源**: Mashable | 2026-01-18  
**链接**: https://mashable.com/article/openai-to-finally-bring-ads-to-chatgpt

OpenAI 确认将在未来几周内开始在 ChatGPT 中测试广告，面向美国的免费用户和 Go 层级用户。Pro、Business 和 Enterprise 付费层级将保持无广告。广告将被清晰标记，与回答分开放置，用户可以关闭广告或完全关闭个性化。这标志着 OpenAI 商业化战略的重大转变，运营成本据报道达到数十亿美元。

**关键信息**:
- 广告将面向美国的免费用户和 Go 层级用户

- 广告将被清晰标记，与回答分开放置
- 用户可以关闭广告或完全关闭个性化
- 18 岁以下账户或敏感话题（健康、政治）附近不会显示广告
- 对话不会被分享或出售给广告商

### 2. Anthropic 发布 Claude Cowork - AI 代理工具

**来源**: Mashable | 2026-01-14  
**链接**: https://mashable.com/article/anthropic-used-mostly-ai-to-build-claude-cowork-tool

Anthropic 推出 Cowork，这是 Claude Code 的非开发者版本，面向日常工作任务。用户可以授予 Claude 访问文件夹的权限，执行组织文件、起草报告、编译电子表格等实际工作。令人惊讶的是，Cowork 的大部分代码由 Claude Code 自己编写。作为研究预览版推出，面向 macOS 上的 Claude Max 订阅者。

**关键功能**:
- 用户授予 Claude 访问计算机上某个文件夹的权限
- 可以执行实际工作：组织文件、起草报告、从截图编译电子表格、清理文档
- 与标准聊天不同，Cowork 具有更强的自主性
- 用户分配任务，Claude 逐步规划和执行，并随时通知进度
- 大部分代码由 Claude Code 自己编写

### 7. Vitalik Buterin 呼吁简化 Ethereum 协议

**来源**: BeInCrypto | 2026-01-18  
**链接**: https://beincrypto.com/vitalik-buterin-ethereum-success-roadmap/

Vitalik Buterin 宣称 Ethereum 的长期生存取决于大幅简化其协议。他警告过度的技术复杂性威胁其基本主权，并重申"walkaway test"（离开测试）作为成功的关键基准。Buterin 呼吁在开发过程中建立明确的"垃圾回收"功能，删除过时的代码和依赖项。2026 年被定位为 Ethereum 恢复自主权和去信任的关键年。

**关键观点**:
- 依赖"博士级密码学"和日益臃肿的代码可能会缩小 Ethereum 的可访问性
- 网络可能会偏向技术官僚模式，而不是保持去中心化的公共产品
- 重申"walkaway test"（离开测试）作为成功的关键基准
- 该测试衡量如果原始创始人和核心研究人员永久离开项目，区块链是否能继续安全运行
- 呼吁在开发过程中建立明确的"垃圾回收"功能

**三个具体指标**:
1. 最小化总协议代码
2. 减少对复杂组件的依赖
3. 增加自给自足的不变量数量
